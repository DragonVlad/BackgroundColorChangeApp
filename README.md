# Background Color Change Project 

Этот пет-проект представляет собой простое веб-приложение, позволяющее пользователям динамически изменять цвет фона страницы по нажатию клавиш. Он демонстрирует использование
функций, обработку событий и управление стилями на JavaScript.

### Основные возможности:
- Динамическое переключение цвета фона страницы;
- Обработка событий нажатия клавиш;
- Использование случайности выбора цвета.

### Использование:
Для смены цвета фона страницы нажмите любую клавишу на клавиатуре.
